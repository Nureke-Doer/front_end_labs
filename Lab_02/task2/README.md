## Overview
This laboratory work demonstrates the use of React Fragments, JSX expressions, and component composition.

## Lab 2.1 Components
- **FragmentLayout:** A layout using fragments to wrap header, main, and footer without extra DOM nodes.
- **ItemList:** Maps an array of items to a list using unique keys.
- **Combined Component:** Displays the title, list, and a summary with the total count.

## Lab 2.2 Components
- **Card:** A reusable component that accepts `title` and `children` props.
- **ProductList:** Maps a product array into several Card components.
- **Section:** A fragment-based wrapper with a title.
